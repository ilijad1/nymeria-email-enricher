Upload Document

Supported File Formats: (.csv, .csvz, .tsv .xlsx, .xlsm, .ods)

© Psyent 2021

Steps:

1. Choose a comma separated value document (see supported formats).

2. Click Find Emails.

3. Wait for the application to process the emails.




Rules:

- The application will modify the existing document and return it after processing.

- The column that contains the LinkedIn URLs must be set to the following name: linkedinUrl.

- The column that will hold the emails must be set to the following name: email.

- The order of the columns does not matter.

Example Doc:
Full Name linkedinUrl email
John Doe https://www.linkedin.com/in/john-doe (empty)
